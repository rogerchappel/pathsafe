# Roadmap

## Near-term

- Publish `0.1.x` after initial user feedback.
- Add more Windows path fixtures in CI.
- Improve glob documentation for edge cases.
- Add structured error helpers for CLI integrations.

## Later

- Optional async API variants.
- Optional case-sensitivity probes and warnings.
- More examples for coding-agent file tools.
- Policy presets for read-only and write-safe workflows.

## Non-goals

- Kernel-level sandboxing.
- Replacing OS permissions.
- Hiding filesystem race conditions from callers.
